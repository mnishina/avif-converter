# AVIF Image Converter

PNG/JPEG画像をAVIF形式に一括変換するCLIツールです。
Figmaなどのデザインツールから書き出した画像を効率的に変換することを目的としています。

## 特徴

- 📁 フォルダ内の画像を一括変換（サブディレクトリ構造を維持）
- 🔄 同形式の圧縮版（フォールバック用）を同時生成
- 🧹 変換前に出力フォルダを自動リセット
- ⚙️ 画質・エンコード品質の詳細設定
- 💬 対話形式のインタラクティブモード
- 🚀 マルチコアを活用した高速並列処理

## インストール

```bash
npm install
```

## 使い方

### インタラクティブモード

引数なしで実行すると、対話形式で設定を入力できます。

```bash
npm run convert
```

### コマンドライン引数モード

引数を指定して実行することで、自動化やスクリプトへの組み込みが可能です。

```bash
# 基本的な使い方
npm run convert -- -i ./input -o ./output -q 80

# 短縮形
npm run convert -- -i ./input -o ./output -q 80
```

## オプション一覧

| 引数 | 短縮形 | 説明 | デフォルト値 |
|------|--------|------|-------------|
| `--input` | `-i` | 入力フォルダのパス | `./input` |
| `--output` | `-o` | 出力フォルダのパス | `./output` |
| `--quality` | `-q` | 画質 (0-100) | `80` |
| `--effort` | `-e` | エンコード品質 (0-9) | `4` |
| `--resize` | `-r` | リサイズ (例: 50%, 1920) | - |
| `--pattern` | `-p` | ファイルパターン (glob) | `*.{png,jpg,jpeg,webp}` |
| `--yes` | `-y` | 確認プロンプトをスキップ | `false` |
| `--silent` | `-s` | 進捗表示を最小化 | `false` |
| `--help` | `-h` | ヘルプを表示 | - |
